simple_term_menu 坑人啊，他不支持 windows！（用了 termios ，这个库不支持 windows ）

simple_term_menu不支持windows这个问题，我在网上搜，好多年之前就有人提出这个问题了，但是作者有回应，但也没有修复这个问题。 下面展示，这个，能够在 windows 上运行的 meun 函数，同时兼容 windows 和 macos ！

为此，我需要研究一下如何获取键盘输入，然后自己写一个简单的 menu 函数。然后又踩坑了，有用上了智能适配windows的 msvcrt.getch() 最终我弄出来了这个函数：

 1def get_key(): #get keypress using getch , not support F1-F10 and esc and tab and so on ....
 2    first_char = getch.getch()
 3    if first_char == '\x1b': #arrow keys
 4        a=getch.getch()

 7    if ord(first_char) == 10:
 8        return  'enter'

10        return  'space'
11    else:
12        return first_char #normal keys like abcd 1234

14while True:

这个函数可以获取键盘输入，返回值是一个字符串，比如up down left right enter space a b c d 1 2 3 4等等。

menu代码

当然写这个meun我觉得还是挺难的）我拷问了chatgpt老半天，他才给我编好（最后的输入编码还是我自己解决的，不过这个快捷键识别功能是他写好的）

使用方法：传入otpions（一个list），可以在里面加入快捷键，比如[1]Option 1，然后就可以通过1来选择这个选项。也可以通过键盘上下来控制选择的项目。

 1import os
 2import re
 3import getch
 4
 5def get_key(): #get keypress using getch , not support F1-F10 and esc and tab and so on ....
 6    first_char = getch.getch()
 7    if first_char == '\x1b': #arrow keys
 8        a=getch.getch()
 9        b=getch.getch()
10        return {'[A': 'up', '[B': 'down', '[C': 'right', '[D': 'left' }[a+b]
11    if ord(first_char) == 10:
12        return  'enter'
13    if ord(first_char) == 32:
14        return  'space'
15    else:
16        return first_char #normal keys like abcd 1234
17
18def scan_short_cuts(options):
19    shortcuts = {}
20    for i, option in enumerate(options):
21        match = re.match(r"\[(.*)\](.*)", option)
22        if match:
23            shortcut, text = match.group(1, 2)
24            shortcuts[bytes(shortcut,'utf-8')] = i
25    return shortcuts
26
27
28def show_menu(options, selected_index):
29    os.system("cls" if os.name == "nt" else "clear")
30    print("Menu","current option:",selected_index)
31    for i, option in enumerate(options):
32        if i == selected_index:
33            print(f"> {option}")
34        else:
35            print(f"  {option}")
36    print("\nUse the arrow keys to move, Enter to select, and Esc to exit.")
37
38def get_menu_choice(options):
39    shortcuts = scan_short_cuts(options)  # scan for shortcuts
40    selected_index = 0
41    while True:
42        show_menu(options, selected_index)
43        key = get_key()
44        if key == 'enter':  # Enter key to select
45            return selected_index
46        elif key in ('up','down'):  # Up or Down arrow
47            selected_index = (selected_index + (1 if key == 'down' else -1) + len(options)) % len(options)
48        elif key in shortcuts:  # Shortcut key
49            return shortcuts[key] + 1

使用的话，只需要：

思路

其实我一开始想要参考一下 simple_term_menu 的源码，但是我发现我太菜了，根本看不懂他们的乱七八糟，贼复杂的源代码，所以我就自己写了一个简单的。

传入一个list，get_menu_choice(options) ,通过正则表达式识别快捷键存储到字典里，然后通过 show_menu(options, selected_index) 循环打印出来(每次打印之前清屏)。

然后通过get_key()获取键盘输入，不断循环！ ok。
