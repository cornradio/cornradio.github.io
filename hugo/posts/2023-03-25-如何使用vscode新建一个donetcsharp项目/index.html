通常情况下，使用 Visual Studio 来编写 C# 代码创建项目和管理非常的简单。但是，Visual Studio 是一个非常大的 IDE（而且mac版不好用），如果您只是想编写一些简单的 C# 代码，那么它可能会显得有些笨蛋。

以下是如何使用 Visual Studio Code 搭建 C# 环境并输出 “Hello World” 的简单步骤：

安装 .NET Core SDK

首先，需要安装 .NET Core SDK。可以在官网下载安装包进行安装：https://dotnet.microsoft.com/download ，我目前使用.NET 6.0。

安装 C# 插件

然后，在 Visual Studio Code 中搜索安装 C# 插件。按下 ctrl + Shift + x 打开扩展商店，搜索 “C#” 并点击安装。

配置插件

右键插件，选择 “Configure Extension Settings”，在弹出的窗口中增加搜索关键词 sdk ，修改sdk位置（是上面你刚刚装好的路径）

新建项目

ctrl + ` 打开终端。在终端中输入以下命令：

1mkdir HelloWorld
2cd HelloWorld
3dotnet new console

这将创建一个名为 “HelloWorld” 的文件夹，并在其中生成一个新的 .NET Core 控制台应用程序。（文件结构蛮复杂的）

他会自动生成一个 Program.cs 文件，hello world的代码就在这里面。
